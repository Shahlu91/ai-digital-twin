# Shahzaib Mian - AI Digital Twin

Standalone HTML version - No build needed!

## 🚀 Deploy to Vercel (5 minutes)

### Step 1: Get API Key
1. Go to https://console.anthropic.com/
2. Create/Login → API Keys → Create Key
3. Copy the key (starts with `sk-ant-...`)

### Step 2: Edit index.html
1. Open `index.html` in a text editor
2. Find line with: `const apiKey = '{VITE_ANTHROPIC_API_KEY}';`
3. Replace `{VITE_ANTHROPIC_API_KEY}` with your actual API key
4. Save the file

### Step 3: Deploy to Vercel
1. Go to https://vercel.com/new
2. Upload the folder (just drag & drop)
3. Click "Deploy"
4. Done! 🎉

Your URL: `https://your-project.vercel.app`

## 📝 Files
- `index.html` - Complete standalone app (React via CDN)
- `vercel.json` - Vercel config

## 🔒 Security Note
Your API key will be visible in the HTML source. For production, use environment variables instead.
